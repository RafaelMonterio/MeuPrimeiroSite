Oque é html?

HTML é a linguagem de marcação padrão para paginas web.

  • Significa Linguagem de marcação de hipertexto.
  • É a linguagem de marcação padrão para paginas web.
  • Descreve a estrutura de uma pagina web
  • Consiste de uma serie de elementos.
  • Elementos dizem para o navegador como exibir o conteúdo.
  • OS elementos rotulam pedaços do conteúdo como "Isso é um Título", "Isso é um paragrafo", "Isso é um link".

Oque é CSS?

CSS é a linguagem que nos usamos para estilizar uma pagina web

  • Significa Folhas de estilo em cascata
  • Descreve como os elementos do HTML vão ser exibidos no site, no papel, ou em outra midia.
  • Economisa muito trabalho. Ele pode controlar o layout de multiplas paginas web de uma vez.
  • Paginas de estilo exteno são armazenadas em arquivos CSS.

Oque é JS?

JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

  • JavaScript É facil de aprender.
